## Description

Coursework for [NestJS Zero To Hero](https://www.udemy.com/course/nestjs-zero-to-hero) with Ariel Weinberger. Develop and deploy enterprise back-end applications following best practices using Node.js and TypeScript.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
